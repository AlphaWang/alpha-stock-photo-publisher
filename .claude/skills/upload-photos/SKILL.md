# upload-photos

Upload stock photos to Shutterstock and/or 500px.com.cn using metadata generated by `photo_desc.py`.

## Usage

```
/upload-photos <directory> [--platform shutterstock|px500|all] [--dry-run]
```

- `<directory>` — folder containing images and their `*_*.json` metadata files
- `--platform` — which platform(s) to upload to (default: `all`)
- `--dry-run` — preview which files would be uploaded without actually uploading

## Examples

```
/upload-photos ~/Photo/2026-04-15/4
/upload-photos ~/Photo/2026-04-15/4 --platform shutterstock
/upload-photos ~/Photo/2026-04-15/4 --dry-run
```

## What this skill does

1. Parse `<directory>` and `--platform` from the user's message
2. Run: `python3 upload_photos.py <directory> --platform <platform> [--dry-run]`
3. The script will open a browser window if login is required — guide the user to log in and press Enter
4. Report the success/failure summary from the script output

## Prerequisites

```bash
pip install playwright
playwright install chromium
```

## First run

On the first run for each platform, a browser window will open. Log in with your account credentials (including any 2FA), then press Enter in the terminal. The session is saved and reused on future runs.
